# קרב עב״אמים

משחק Web תלת ממדי בעברית: שולטים בחייזר ירוק, מפילים חלליות עוינות, ולבסוף בוחרים סגל ומגלים מי יום הקרב הבא.

## פיתוח מקומי

```powershell
npm run dev
```

פותחים בדפדפן:

```text
http://127.0.0.1:5177
```

## דריסה (Deploy) ל-Vercel

הפרויקט הוא סטטי לחלוטין (HTML + CSS + JS, ללא בנייה). העלאה ל-Vercel:

```powershell
# התחברות חד-פעמית
npx vercel login

# פריסה ראשונה (מתוך ספריית הפרויקט)
npx vercel

# פריסה לסביבת production
npx vercel --prod
```

או דרך Git: דחפו את הריפו ל-GitHub, התחברו ל-Vercel, חברו את הריפו, ו-Vercel יזהה את `vercel.json` ויפרוס אוטומטית.

## עריכת רשימת השמות

ב-`src/main.js`:

```js
const SEGEL_DATA = {
  "א": {
    names: ["עידן", "דביר", "איה", "מיכאל", "עלמה", "עדי", "אורי", "שאבון"],
    winner: "עדי",
  },
  "ב": {
    names: ["ליה", "ינאי", "זוהר", "איתי", "חן", "ליאור", "רון", "רותם"],
    winner: "ליאור",
  },
};
```

`names` קובע את השמות בגלגל לכל סגל, `winner` קובע על מי הגלגל ינחת בפועל.
